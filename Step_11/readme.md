Z-Buffer 算法()
{
    帧缓存全部置为背景色
    深度缓存全部置为最小的Z值
    for(每一个多边形)
    {
        扫描转换该多边形
        for(该多边形在该像素的深度值Z(x,y))
        {
            计算该多边形在该像素的深度值Z(x,y)
            if( z(x,y) > z缓存在(x,y)的值 )
            {
                把z（x,y）存入 z缓存中(x,y)处
                把多边形在(x,y)处的颜色值存入帧缓存的(x,y)处
            }
        }
    }
}
